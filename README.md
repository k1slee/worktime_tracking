# Система учета рабочего времени сотрудников

## Описание
Веб-приложение для учета рабочего времени сотрудников на предприятии с разделением ролей:
- Администратор
- Мастер
- Плановый отдел

## Технологии
- Python 3.10+
- Django 4.2
- Django REST Framework
- SQLite
